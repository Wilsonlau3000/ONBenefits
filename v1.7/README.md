# Ontario Benefits Finder (Prototype) — EN + 简体中文

## Run locally
Open `index.html` in your browser.

If your browser blocks local JS:
- Python: `python -m http.server 8080`
Then open: http://localhost:8080/ontario-benefits-prototype-i18n/index.html

## Language
Use the language dropdown (top-right). Your choice is saved in localStorage.

## Where to edit
- `i18n.js`: all translations (EN + zh-Hans)
- `engine.js`: benefit list, links, and matching rules (simplified)
- `app.js`: UI rendering
